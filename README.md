## Features

- **Multi-threaded** request execution
- **Random delays** between requests (anti-detection)
- **Rotating User-Agents** from a built-in list of 200+ real browsers & bots
- Real-time console feedback with emojis
- Comprehensive logging to file
- Final summary statistics

- Python 3.6 or higher
- `requests` library
